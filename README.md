# osu!türkiye Wiki

Bu repository, osu!türkiye topluluğunun wiki sayfalarını içerir.

## Yapı

- `index.json`: wiki manifest dosyası
- `wiki/*.md`: markdown sayfaları

## Sayfa bağlantıları

Wiki sayfaları, `osutr-web` içinde temiz slug yapısıyla yayınlanmak üzere hazırlanır.
Örnek sayfa adresleri:

- `/wiki/anasayfa`
- `/wiki/hakkimizda`
- `/wiki/topluluklar`
- `/wiki/etkinlikler`
- `/wiki/ekip`
